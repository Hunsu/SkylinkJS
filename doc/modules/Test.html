Running test scripts

Installing required dependencies

You will require to install the following dependencies on the npm. SkylinkJS test relies on tape and testling. Install these required modules. Make sure you have forked the SkylinkJS github repo.

      npm install tape
      npm install testling

After you have installed the required dependencies, simple go to your terminal and enter the command lines available. 

    sh test.sh (#test)

List of available Tests

Each test is denoted in this format for each class or module. (#class)_Test or (#module)_Test
