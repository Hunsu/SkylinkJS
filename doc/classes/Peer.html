Defined in: source/Peer.js:146

Access level: private

Stores the streaming configuration for the peer connection.

Object Keys:
  • audio : type JSON | Boolean

    value: default = false

    This argument is optional.

    The audio stream configuration. If parsed as a boolean, other configuration settings under the audio configuration would be set as the default setting in the connection.

    • stereo : type Boolean

      value: default = false

      This argument is optional.

      The flag that indiciates if stereo is enabled for this connection.

    • sourceId : type String

      This argument is optional.

      The source id of the audio MediaStreamTrack used for this connection.

  • video : type String | Boolean

    value: default = false

    This argument is optional.

    The video stream configuration. If parsed as a boolean, other configuration settings under the video configuration would be set as the default setting in the connection.

    • resolution : type JSON

      This argument is optional.

      The video streaming resolution.

      • width : type Integer
        The video resolution width.
      • height : type Integer
        The video resolution height.
    • frameRate : type Integer

      The video stream framerate.

    • sourceId : type String

      This argument is optional.

      The source id of the video MediaStreamTrack used for this connection.

  • status : type JSON

    The stream MediaStreamTrack status.

    • audioMuted : type Boolean

      value: default = false

      This argument is optional.

      The flag that indicates if audio is muted. If audio is set to false, this would be set as true.

    • videoMuted : type Boolean

      value: default = false

      This argument is optional.

      The flag that indicates if video is muted. If video is set to false, this would be set as true.
