init

(
  • options
)

Defined in source/skyway.js:764

IMPORTANT: Please call this method to load all server information before joining the room or doing anything else. The Init function to load Skyway. If you would like to set the start time and duration of the room, you have to generate the credentials. Steps to generate the credentials:

  • Hash: This hash is created by using the roomname, duration and the timestamp (in ISO String format).
  • E.g: hash = CryptoJS.HmacSHA1(roo

Parameters:

  • options Object

    Connection options or API Key ID [init('API_KEY')]

    • roomserver String

      mname + '' + duration + '' + (new Date()).toISOString()).

      • Credentials: The credentials is generated by converting the hash to a Base64 string and then encoding it to a URI string.
      • E.g: encodeURIComponent(hash.toString(CryptoJS.enc.Base64))
    • apiKey String

      API Key ID to identify with the Temasys backend server

    • defaultRoom String

      Optional. The default room to connect to if there is no options.room provided.

    • room String

      Optional. The room joinRoom connects to.

    • roomServer String

      Optional. Path to the Temasys backend server If there's no room provided, default room would be used.

    • region String

      Optional. The regional server that user chooses to use. [Rel: Skyway.REGIONAL_SERVER]

    • iceTrickle String

      Optional. The option to enable iceTrickle or not. Default is true.

    • credentials String

      Optional. Credentials options

      • startDateTime String
        The Start timing of the meeting in Date ISO String
      • duration Integer
        The duration of the meeting
      • credentials String
        The credentials required to set the timing and duration of a meeting.

Example:

SkywayDemo.init('API_KEY');

SkywayDemo.init({ 'apiKey' : 'API_KEY', 'roomServer' : 'ROOM_SERVER', 'defaultRoom' : 'CAT_FORUM', 'room' : 'PERSIAN_CATS' });

SkywayDemo.init({ 'apiKey' : 'API_KEY', 'roomServer' : 'ROOM_SERVER', 'defaultRoom' : 'CAT_FORUM', 'room' : 'PERSIAN_CATS', 'credentials' : { 'startDateTime' : (new Date()).toISOString(), 'duration' : 500, 'credentials' : 'THE_CREDENTIALS' } });
