Self ( config ,   listener ,   ) Version 0.6.0

Defined in: source/Self.js:1

This Self constructor cannot be instantiable, meaning you cannot create a new Self class object that would require certain parent dependencies functionalities. E.g. DataChannel class relies on Peer class.

To create a Self class object, these are the required parameters to be provided.

Parameters:
  • config : type JSON

    The user configuration.

    • id : type String

      The self user id.

    • username : type String

      The self user's username received from API server.

    • timeStamp : type String

      The self user's timestamp (ISO format) received from API server.

    • token : type String

      The self user's token received from API server.

    • data : type String

      This argument is optional.

      The self user's custom data.

  • listener : type Function

    The listener function.

getInfo ( [peerId] ,   ) Version 0.6.0

Defined in: source/Self.js:279

Gets the self user info.

Parameters:
  • peerId : type String

    This argument is optional.

    The shared peer connection id. If no shared peer connection id is provided, it will return as "stream" instead of "streams".(#peerId).

Returns:
  • result : type JSON

    The self user streaming configuration and custom data.

    • userData : type String | JSON
      The custom data.
    • agent : type JSON
      The user's browser agent information.
    • agent.name : type String
      The user's browser agent name.
    • agent.version : type Integer
      The user's browser agent version.
    • agent.webRTCType : type String
      The user's browser webrtc implementation type.
    • streams : type JSON
      The list of peer connections streaming.
    • streams.(#peerId) : type JSON
      The peer connection streaming information.
    • streams.(#peerId).audio : type JSON|Boolean
      The audio streaming information. If there is no stream connection with the peer, it's false.
    • streams.(#peerId).audio.stereo : type Boolean
      The flag that indicates if stereo is enabled for this connection. By default, it's false.
    • streams.(#peerId).audio.sourceId : type String
      The audio MediaStreamTrack source used for this connection.
    • streams.(#peerId).video : type JSON|Boolean
      The video streaming information. If there is no stream connection with the peer, it's false.
    • streams.(#peerId).video.resolution : type JSON
      The video stream resolution.
    • streams.(#peerId).video.resolution.width : type Integer
      The video stream resolution height.
    • streams.(#peerId).video.resolution.height : type Integer
      The video stream resolution width.
    • streams.(#peerId).video.frameRate : type Integer
      The video stream resolution framerate.
    • streams.(#peerId).video.sourceId : type String
      The video MediaStreamTrack source used for this connection.
    • streams.(#peerId).status : type JSON
      The MediaStreamTracks enabled status (muted/unmuted).
    • streams.(#peerId).status.audioMuted : type Boolean
      The audio MediaStreamTrack enabled status (muted/unmuted).
    • streams.(#peerId).status.audioMuted : type Boolean
      The video MediaStreamTrack enabled status (muted/unmuted).
    • bandwidth : type JSON
      The bandwidth configuration for the peer connections. This does fixes the bandwidth but doesn't prevent alterations done by browser for smoother streaming.
    • bandwidth.data : type Integer
      The bandwidth configuration for data stream.
    • bandwidth.video : type Integer
      The bandwidth configuration for video stream.
    • bandwidth.audio : type Integer
      The bandwidth configuration for audio stream.
      // Example for constructor getInfo ()

agent type JSON Version 0.6.0

Defined in: source/Self.js:84

Access level: private

The self user browser agent information.

Object Keys:
  • name : type String

    The user's browser agent name. For other SDKs, it's indicated by their type of device E.g. ios, android. For components, it's indicated by their type E.g. MCU, Recording.

  • version : type Integer

    The user's browser agent version. For other SDKs, it's indicated by their version of device OS ios8 = 8. android kitkat = 4.4. For components, it's indicated by their version E.g. 0.1.0, 0.2.0.

  • webRTCType : type String

    The user's browser agent webrtc implementation type. For other SDKs or components, use other.
