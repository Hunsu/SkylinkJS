Constructor

User ( config ,   listener ,   ) Version 0.6.0

Defined in: source/User.js:1

This User constructor cannot be instantiable, meaning you cannot create a new User class object that would require certain parent dependencies functionalities. E.g. DataChannel class relies on Peer class.

To create a User class object, these are the required parameters to be provided.

Parameters:
  • config : type JSON

    The user configuration.

    • id : type String

      The user id.

    • agent : type JSON

      The user browser agent information.

      • name : type String
        The user browser agent name.
      • version : type Integer
        The user browser version.
      • webRTCType : type String
        The user browser WebRTC implementation type.
    • data : type JSON | String

      The user custom data.

    • stream : type JSON

      The streamming configuration for the "main" shared peer connection.

      • audio : type JSON | Boolean

        value: default = false

        This argument is optional.

        The audio stream configuration. If parsed as a boolean, other configuration settings under the audio configuration would be set as the default setting in the connection.
      • audio.stereo : type Boolean

        value: default = false

        This argument is optional.

        The flag that indiciates if stereo is enabled for this connection.
      • audio.sourceId : type String

        This argument is optional.

        The source id of the audio MediaStreamTrack used for this connection.
      • video : type String | Boolean

        value: default = false

        This argument is optional.

        The video stream configuration. If parsed as a boolean, other configuration settings under the video configuration would be set as the default setting in the connection.
      • video.resolution : type JSON

        This argument is optional.

        The video streaming resolution.
      • video.resolution.width : type Integer
        The video resolution width.
      • video.resolution.height : type Integer
        The video resolution height.
      • video.frameRate : type Integer
        The video stream framerate.
      • video.sourceId : type String

        This argument is optional.

        The source id of the video MediaStreamTrack used for this connection.
      • status : type JSON
        The stream MediaStreamTrack status.
      • status.audioMuted : type Boolean

        value: default = false

        This argument is optional.

        The flag that indicates if audio is muted. If audio is set to false, this would be set as true.
      • status.videoMuted : type Boolean

        value: default = false

        This argument is optional.

        The flag that indicates if video is muted. If video is set to false, this would be set as true.
    • bandwidth : type JSON

      The bandwidth configuration for the peer connections. This does fixes the bandwidth but doesn't prevent alterations done by browser for smoother streaming.

      • audio : type Integer

        This argument is optional.

        The bandwidth configuration for the audio stream.
      • video : type Boolean

        This argument is optional.

        The bandwidth configuration for the video stream.
      • data : type Boolean

        This argument is optional.

        The bandwidth configuration for the data stream.
    • streamObject : type Stream

      This argument is optional.

      The stream object passed for the "main" peer connection.

    • SDPType : type String

      The session description type that the "main" peer connection would send.

  • listener : type Function

    The listener function.
